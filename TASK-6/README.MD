Project Overview

CodeSprint Bootcamp is a responsive landing page designed for a fictional coding bootcamp. The website showcases courses, student achievements, testimonials, and contact information through a modern and user-friendly interface.

Features

- Responsive Navigation Bar
- Hero Section with Call-to-Action Button
- Courses Offered Section
- Student Statistics Section
- Testimonials Section
- Contact Form
- Form Validation using JavaScript
- Mobile-Friendly Responsive Design

Technologies Used

- HTML5
- CSS3
- JavaScript

Learning Outcomes

- Responsive Web Design
- CSS Flexbox and Layout Techniques
- JavaScript Event Handling
- Form Validation
- UI/UX Design Principles
- Building Professional Landing Pages
