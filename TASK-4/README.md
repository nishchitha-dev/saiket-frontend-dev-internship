Task 4 - Quiz Application

Project Overview

This project is an interactive Quiz Application developed using HTML, CSS, and JavaScript. The application allows users to answer multiple-choice questions, calculate scores, and view their final results.

Features

- Multiple-choice questions
- Interactive user interface
- Score calculation
- Instant result display
- Responsive design
- User-friendly experience

Technologies Used

- HTML5
- CSS3
- JavaScript

Learning Outcomes

- Understanding DOM Manipulation
- Handling User Events
- Working with JavaScript Functions
- Managing User Input
- Building Interactive Web Applications
