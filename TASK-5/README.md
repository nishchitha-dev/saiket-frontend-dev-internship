Task 5 - To-Do List Application

Project Overview

This project is a To-Do List Application developed using HTML, CSS, and JavaScript. The application helps users manage daily tasks by allowing them to add, complete, and delete tasks through a simple and interactive interface.

Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Interactive user interface
- Responsive design
- Easy task management

Technologies Used

- HTML5
- CSS3
- JavaScript

Learning Outcomes

- DOM Manipulation
- Event Handling
- Dynamic Content Updates
- JavaScript Functions
- User Interface Development
